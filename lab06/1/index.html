ภาษาโปรแกรม (Programming Language) คือ ภาษาที่ใช้สื่อสารกับคอมพิวเตอร์ เพื่อให้คอมพิวเตอร์ทำงานตามที่เราต้องการ ภาษาโปรแกรมมีหลากหลายประเภท แต่ละประเภทมีจุดเด่นและจุดด้อยแตกต่างกันไป การเลือกภาษาโปรแกรมจึงขึ้นอยู่กับความต้องการของเราว่าต้องการใช้คอมพิวเตอร์ทำอะไร
ภาษา C เป็นภาษาโปรแกรมระดับสูงที่ได้รับความนิยมสูงในปัจจุบัน ภาษา C ถูกพัฒนาขึ้นโดย Dennis Ritchie ที่ห้องปฏิบัติการ Bell Labs ในช่วงปี ค.ศ. 1969-1973 ภาษา C เป็นภาษาที่มีความยืดหยุ่นสูง ออกแบบมาให้สามารถทำงานกับคำสั่งพื้นฐานของคอมพิวเตอร์ได้อย่างมีประสิทธิภาพ
ภาษา Python เป็นภาษาโปรแกรมระดับสูงที่ได้รับความนิยมสูงในปัจจุบัน ภาษา Python ถูกพัฒนาขึ้นโดย Guido van Rossum ในช่วงปี ค.ศ. 1989-1991 ภาษา Python เป็นภาษาที่เขียนง่าย เหมาะสำหรับงานต่างๆ มากมาย เช่น การพัฒนาซอฟต์แวร์ การวิเคราะห์ข้อมูล ปัญญาประดิษฐ์ เป็นต้น
ภาษา Java เป็นภาษาโปรแกรมเชิงวัตถุ (Object-Oriented Programming) พัฒนาโดย James Gosling และวิศวกรคนอื่นๆ ที่บริษัท Sun Microsystems ภาษาจาวาถูกพัฒนาขึ้นในปี พ.ศ. 2534 โดยเป็นส่วนหนึ่งของ โครงการกรีน และสำเร็จออกสู่สาธารณะในปี พ.ศ. 2538
